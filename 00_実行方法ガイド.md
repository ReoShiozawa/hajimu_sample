# 作業ガイド - はじむサンプルコード集の実行方法

## 📋 始める前に

### 必須
- はじむ言語がインストールされていること
- ターミナル/コマンドプロンプトへのアクセス

### オプション（特定のサンプル実行時）
- hajimu_web（Webサーバーサンプル用）
- hajimu_discord（Discord Botサンプル用）
- hajimu_gui（GUIサンプル用）

## 🚀 基本的な実行方法

### 最小限から始める

```bash
cd hajimu_sample

# 最初のサンプルを実行
hajimu 01_基本文法/01_hello_world.jp
```

**出力:**
```
こんにちは、はじむ世界！
```

### ステップバイステップ実行

```bash
# 1. 基本文法を学ぶ（5～10分）
hajimu 01_基本文法/02_variables_and_types.jp
hajimu 01_基本文法/03_arithmetic_operations.jp
```

## 🌐 Web開発（hajimu_web）

### セットアップ

```bash
# hajimu_webプラグインをインストール
hajimu パッケージ 追加 ReoShiozawa/hajimu_web
hajimu パッケージ インストール
```

### サンプル実行

```bash
# 最小HTTPサーバー
hajimu 06_Web開発（hajimu_web）/01_hello_server.jp
# ブラウザで http://localhost:8080 にアクセス

# JSON APIサーバー
hajimu 06_Web開発（hajimu_web）/02_json_api.jp
# ブラウザで http://localhost:3000 にアクセス
```

**トラブルシューティング:**
```
ポートが既に使用されている場合は、コード内のポート番号を変更してください：
ウェブ.サーバー作成(9000)  # 8080から9000に変更
```

## 🤖 Discord Bot開発（hajimu_discord）

### セットアップ

```bash
# 1. hajimu_discordプラグインをインストール
hajimu パッケージ 追加 ReoShiozawa/hajimu_discord
hajimu パッケージ インストール

# 2. Discord Developer Portal から Bot トークンを取得
# https://discord.com/developers/applications

# 3. 環境変数を設定
export DISCORD_TOKEN="your_bot_token_here"
```

### サンプル実行

```bash
# 最小Discord Bot
DISCORD_TOKEN=your_token hajimu 07_Discord_Bot（hajimu_discord）/01_hello_bot.jp
```

**機能テスト（あなたの Discord サーバーで）:**
```
!ping          # ポン！と返答
!hello         # こんにちは！と返答
!info          # 埋め込みメッセージを送信
/greet         # スラッシュコマンド
/add           # 計算結果を返します
```

## 🐛 トラブルシューティング

### `hajimu: command not found`
```bash
# パスの確認
which hajimu

# または直接実行
/usr/local/bin/hajimu 01_基本文法/01_hello_world.jp
```

### パッケージエラー
```bash
# インストール済みパッケージ確認
hajimu パッケージ 一覧

# キャッシュクリア
rm -rf ~/.hajimu/cache
hajimu パッケージ インストール
```

### ポート競合エラー
```
"Address already in use" エラーが出た場合は、
別のポート番号を使用してください：
ウェブ.サーバー作成(9000)
```
